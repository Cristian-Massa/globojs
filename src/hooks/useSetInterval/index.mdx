# useSetInterval

This hook allows you to set an interval and get the number of cicles that have passed.

### Example :

```tsx
	const cicles = useSetInterval(args.miliseconds, args.callback)

	return (
			<Button className="font-bold">Cicles: {cicles}</Button>
	)
```
