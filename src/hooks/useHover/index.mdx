# useHover

This hook allows you to detect when the mouse is hovering over an element.

### Example :

```tsx
	const ref = useRef<HTMLButtonElement>(null)
	const isHovered = useHover(ref)
	return (
		<Button ref={ref} className="w-40 font-bold">

		</Button>
	)
```
