# useSetTimeout

This hook allows you to set a timeout and get a boolean value when the timeout is done.

### Example :

```tsx
	const isDone = useSetTimeout(args.miliseconds, args.callback)
	return <Button>{isDone ? 'Done' : 'Not done'}</Button>
```
