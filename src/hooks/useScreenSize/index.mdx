# useScreenSize

This hook is used to get the screen size of the window. It returns an object with the height and width of the screen.
SSR compatible.

### Example :

```tsx
    const { height, width} =useScreensize()
	return (
		<div className="flex flex-col items-center gap-4">
            <p>
                Height: {height}
            </p>
            <p>
                Width: {width}
            </p>
		</div>
	)
```
