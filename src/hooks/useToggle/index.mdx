# useToggle

This hook is used to toggle between two states. It returns an object with the current state and a function to toggle the state.

### Example :

```tsx
	const {toggle, handleToggle } = useToggle()
	return <Button onClick={handleToggle}>{JSON.stringify(toggle)}</Button>
```
